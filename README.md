# Personal Website - Adel Ghaedi

A personal resume website to showcase skills, projects, and contact information.

## Features

* Responsive design using **Bootstrap 5**
* Main sections:

  * About Me
  * Skills
  * Projects
  * Contact
* Built with **Django 4**
* Professional icons and clean user experience

## Installation & Run

1. Clone the repository:

```bash
git clone https://github.com/adelghaedi/resume-website.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the server:

```bash
python manage.py runserver
```

4. Open the browser at:

```
http://127.0.0.1:8000/
```

## Technologies

* Python / Django
* Bootstrap / HTML / CSS / JavaScript
* PostgreSQL
